How to Make a report

See How to set up plots to get started

Set up a report template

In report_templates there are templates that can be copied and re-named to make a report.

design_template = "generic_report_template.jmd"

Generate report

The report can be generated as a LaTeX PDF (default) or an html file. To set the document type as HTML, use the key word argument doctype = "md2html"

out_path = joinpath(pwd(), "Generated_Reports/")
if !isdir(out_path)
    mkdir(out_path)
end
report(results, out_path, design_template; doctype = "md2html")

creates an HTML

report(results, out_path, design_template)

creates a PDF
